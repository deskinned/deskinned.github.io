Make GitHub yours.

GitSkin is a browser extension that skins GitHub's entire UI. Pick a theme from the marketplace, build your own, or just slap some colors on it and call it a day.

Add to Chrome Browse Themes

Semantic theming

Themes target abstract components, not brittle CSS selectors. When GitHub changes their DOM, your theme survives.

Live adapters

Adapters resolve GitSkin's component model to GitHub's actual DOM. They're served from a CDN, so the extension never needs an update when GitHub ships changes.

5 levels of control

Level 0: just override Primer color tokens. Level 4: target individual components with scoped CSS. Pick your depth.
